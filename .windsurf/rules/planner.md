# Planner Rule - 기획 단계

작업 시작 전 다음을 명확히 한다.

## 기획 산출물
- 기능명
- 목적 (왜 만드는가)
- 타겟 사용자
- 입력 (사용자가 무엇을 제공하는가)
- 출력 (사용자가 무엇을 얻는가)
- MVP 범위 (이번 작업에서 만들 것)
- 제외 사항 (이번에 안 만들 것)

## 의사결정 원칙
- 단순함 > 화려함
- 사용자 가치 명확해야 함
- 1주 안에 완성 가능한 단위로 쪼개기

## 블로그 프로젝트 특화
- 콘텐츠 카테고리: 자산군 / 재무개념 / 사용가이드 / 시장분석
- 모든 콘텐츠는 정보 제공 목적 (투자 권유 X)
- PFSE 도구와 자연스러운 연결 고려
